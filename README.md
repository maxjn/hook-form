### React Form With Hooks :)

**React** Form with **hooks** .

See the [Live Project](https://weather-forcast.pages.dev/).

###### Technologies

- HTML,CSS
- ES6
- React
- useState & useEffects Hooks
- custom hooks
- **uuid** package

###### Features

- Generating a uuid for every user

## Installing / Getting started

```shell
npm install

npm run start
```
