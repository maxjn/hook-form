### React Form With Hooks :)

**React** Form with **hooks** .

See the [Live Project](https://maxjn-hook-form.pages.dev).

###### Technologies

- HTML,CSS
- ES6
- React
- useState & useEffects Hooks
- custom hooks
- **uuid** package

- Generating a uuid for every user

## Installing / Getting started

npm install
